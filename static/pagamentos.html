Adicionar Pagamento

Informe somente os dados necessários: número do pedido, valor pago, forma de pagamento e data. O sistema busca o UUID do orçamento pelo número do pedido e salva o pagamento no banco.

Digite exatamente o número do pedido cadastrado no orçamento.
Use vírgula ou ponto para os centavos.
Observação: é necessário estar logado para enviar o token de autorização salvo no navegador. A API usa o número do pedido para buscar o UUID na tabela orcamentos e gravar esse UUID em pagamentos.orcamentoid.
