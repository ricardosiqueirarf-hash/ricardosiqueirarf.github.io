Adicionar Pagamento

Preencha os campos conforme a tabela pagamentos: id int8, created_at timestamptz, orcamentoid uuid e pagamentos jsonb.

O banco gera este valor automaticamente ao inserir.
Opcional. Se ficar vazio, o banco usa o horário padrão da tabela.
Cole aqui o UUID do orçamento que será gravado na coluna orcamentoid.
Informe um JSON válido. Este conteúdo será salvo diretamente na coluna pagamentos do tipo jsonb.
Observação: é necessário estar logado para enviar o token de autorização salvo no navegador. O envio usa o formato do banco: { orcamentoid, pagamentos }; id e created_at só são enviados quando preenchidos e editáveis no futuro.
