# Schemas XSD da NF-e

Coloque aqui os schemas XSD oficiais da NF-e usados para validar os XMLs antes do envio.

Sugestão de organização:

```text
schemas/
  nfe_v4.00/
    leiauteNFe_v4.00.xsd
    tiposBasico_v4.00.xsd
    xmldsig-core-schema_v1.01.xsd
```

Não inventar validação manual. O XML precisa ser validado contra os XSDs oficiais compatíveis com a versão em uso.
