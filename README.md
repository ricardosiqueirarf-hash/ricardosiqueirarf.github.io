# ANODIZA

MVP inicial do sistema SaaS white-label.

Stack inicial:
- Frontend: Next.js + TypeScript
- Backend: FastAPI + Python
- Banco: Supabase/Postgres

Estrutura:
- frontend: telas do usuario
- backend: API central
- database: migracoes do banco

Primeiro escopo:
- cadastro de empresa, loja e usuario
- login
- painel inicial da loja
- estrutura multiempresa com empresa_id e loja_id
